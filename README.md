This project highlights knowledge representation in MeTTa, a logic-based programming language. It features:

- **Family Tree Representation & Queries**
  - `# find-children`
  - `# find-grandchildren`
  
- **List Operations**
  - `# is-member`
  - `# removeDuplicates`
  - `# extracter`
